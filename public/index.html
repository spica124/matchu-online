게임 플레이
1
방 참가 또는 만들기
홈 화면의 방 목록에서 원하는 방을 클릭해 참가하거나, "방 만들기" 버튼으로 직접 방을 개설하세요. 비밀번호 방은 자물쇠 아이콘이 표시됩니다. 빈 슬롯을 클릭하면 초대 링크를 복사할 수 있습니다.
2
게임 시작
호스트만 게임을 시작할 수 있습니다. 시작하면 맵의 문제들이 무작위 순서로 출제됩니다.
3
정답 입력
음악을 듣고 하단 입력창에 정답을 입력한 뒤 엔터를 누르세요. 소문제가 여러 개인 경우 각각 순서대로 맞춰야 합니다. 빠르게 맞출수록 더 높은 점수를 받습니다.
4
초성 힌트 · 스킵
맵 제작자가 설정한 경우 일정 시간 후 초성 힌트가 표시됩니다. 모르겠으면 K 키로 스킵 투표를 할 수 있습니다. 2명 이하는 전원 동의, 3명 이상은 과반수가 스킵하면 다음 문제로 넘어갑니다.
5
점수 계산
맞춘 순서(1등 보너스)와 남은 시간 비율에 따라 점수가 결정됩니다. 모든 문제가 끝나면 최종 순위가 표시됩니다.
맵 제작
1
맵 기본 설정
사이드바에서 "맵 제작"을 클릭합니다. 맵 이름과 카테고리(애니 노래, K-POP, 보컬로이드 등)를 선택하면 소문제 항목이 자동으로 채워집니다.
2
YouTube URL 입력
각 문제에 YouTube 링크를 붙여넣으면 미리보기 플레이어가 나타납니다. 시작 / 끝 슬라이더로 재생 구간을 직접 잡아주세요. 구간이 제한 시간보다 짧으면 자동으로 반복 재생됩니다.
3
소문제 · 정답 입력
각 소문제(노래 제목, 가수 등)의 정답을 입력합니다. 정답이 여러 개인 경우 쉼표(,)로 구분해 입력하면 모두 정답으로 인정됩니다.
4
초성 힌트 설정 (선택)
소문제마다 "초성 힌트" 체크박스를 켜면 몇 초 후에 자동으로 초성이 공개될지 설정할 수 있습니다. 한국어 정답에만 적용됩니다.
5
맵 공개하기
모든 문제를 작성한 뒤 하단의 "맵 공개하기" 버튼을 누르면 맵이 저장되고 바로 방을 만들어 플레이할 수 있습니다. 내 맵 대시보드에서 언제든지 수정하거나 삭제할 수 있습니다.
